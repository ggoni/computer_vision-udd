# Test Images Dataset

This directory contains test images for the computer vision application.

## Images Included

| Filename | Description | Expected Objects |
|----------|-------------|-----------------|
| `street_scene.jpg` | Urban street scene | Cars, people, bicycles, traffic signs |
| `office_desk.jpg` | Modern office workspace | Laptop, mouse, keyboard, coffee cup |
| `kitchen_scene.jpg` | Kitchen with food items | Fruits, vegetables, kitchen utensils |
| `living_room.jpg` | Living room furniture | Couch, pillows, plants, decorative items |
| `cats_couch.jpg` | COCO dataset classic | Cats, remote control, couch |

## Usage

These images are perfect for testing object detection capabilities:

1. **Development Testing**: Use during backend development to test CV models
2. **API Testing**: Upload via the web interface to test the full pipeline  
3. **Model Validation**: Verify detection accuracy with known objects
